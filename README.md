## рҹ“Ӣ Sobre el Proyecto

**ONG Impacta+** es una plataforma SaaS multi-tenant diseГұada para centralizar y automatizar la gestiГіn operativa de Organizaciones No Gubernamentales. Permite administrar eventos, donaciones, ayuda social, rescate ecolГіgico, rifas digitales, recaudaciГіn de fondos, y mucho mГЎs вҖ” todo desde una Гәnica plataforma.

### рҹҺҜ Problema que Resuelve

Las ONGs carecen de sistemas internos de administraciГіn que les permitan gestionar de manera eficiente sus operaciones diarias:

- рҹ“Ӣ GestiГіn de socios, voluntarios y beneficiarios
- рҹҺӘ AdministraciГіn de eventos y recaudaciГіn de fondos
- рҹ’° Control de donaciones y transparencia financiera
- рҹҢҝ OrganizaciГіn de actividades de ayuda social y ecolГіgica
- рҹҺҹпёҸ DigitalizaciГіn de rifas y actividades de fundraising
- рҹҰҺ Registro y estudio de especies nativas (flora y fauna)
- рҹ“Ҡ Seguimiento de poblaciones y evaluaciГіn de impacto ambiental

### рҹ’Ў VisiГіn

Convertirse en la **plataforma lГӯder** para la gestiГіn de ONGs en la regiГіn, ofreciendo una soluciГіn todo-en-uno que permita a las organizaciones enfocarse en su misiГіn social.

## рҹҡҖ InstalaciГіn

### Requisitos Previos

- **Node.js** 20.x LTS
- **pnpm** 8.x+
- **Docker** y **Docker Compose**
- **PostgreSQL** 16.x (o vГӯa Docker)
- **Redis** 7.x (o vГӯa Docker)

### Inicio RГЎpido

```bash
# 1. Clonar el repositorio
git clone https://github.com/tu-usuario/ONG_impacta.git
cd ONG_impacta

# 2. Instalar dependencias
pnpm install

# 3. Configurar variables de entorno
cp .env.example .env
# Editar .env con tus credenciales

# 4. Levantar servicios con Docker
docker compose up -d

# 5. Ejecutar migraciones de base de datos
pnpm db:migrate

# 6. Seed de datos iniciales
pnpm db:seed

# 7. Iniciar en modo desarrollo
pnpm dev
```

### Variables de Entorno

```env
# Base de Datos
DATABASE_URL=postgresql://impacta:password@localhost:5432/impacta

# Redis
REDIS_URL=redis://localhost:6379

# JWT
JWT_SECRET=tu-secreto-super-seguro
JWT_EXPIRES_IN=15m
JWT_REFRESH_EXPIRES_IN=7d

# Pagos
MERCADOPAGO_ACCESS_TOKEN=tu-token
PAYPAL_CLIENT_ID=tu-client-id
STRIPE_SECRET_KEY=tu-secret-key

# Email
SENDGRID_API_KEY=tu-api-key

# App
APP_URL=https://impacta.pinguinoseguro.cl
NODE_ENV=development
```

### Servicios Docker

```bash
# Levantar todos los servicios
docker compose up -d

# Ver logs
docker compose logs -f api

# Detener servicios
docker compose down
```
